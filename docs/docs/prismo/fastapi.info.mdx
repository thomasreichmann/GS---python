PRISMO é uma plataforma que coleta, processa e visualiza dados sobre a saúde dos oceanos, focando em poluição, destruição de habitats, mudanças climáticas. A plataforma é pensada para ser acessível a populações costeiras, empresas e governos para promover uma gestão sustentável dos recursos marinhos.
